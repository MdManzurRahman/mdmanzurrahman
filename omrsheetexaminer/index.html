How it works

Four basic steps to conduct an examination

1

Design OMR Answer Script for MCQ type examination

Design you OMR Answer Script according to your need for the examination. Export it to PDF format and Print

2

Create file containing correct answers

Create a file which contains all the correct answers of all exam sets of a particular examination and save it as *.ans file for further use

3

Read scanned OMR answer scripts

Use any scanner device to scan all the OMR answer scripts and save it all in a particular folder in JPEG format. Read all the answer scripts and save the result as *.res file.

4

Generate result sheet

Use the *.res file to generate customized result sheet. Export it as PDF format and print it as required.

What's New

See the timeline below to get an overview of the project

  • June 2020

    Project Inception and Alpha Version

    Project idea started on a real life situation regarding the high cost of traditional OMR Machine and its maintenance. With a sound background on Artificial Intelligence, Digital Image Processing and Computer Vision a software alternative project idea flourished.

  • July 2020

    Beta Version released

    Several testing needs to be performed on this beta version before launcing fully functional Stable version. No GUI yet

  • August 2020

    Release of V1 after Alpha and Beta testing. GUI added

    Stable version released as Version 1 after Pilot test on Beta version.

  • September 2020

    PDF to JPEG tools added

    Several scanner returns scanned pages as PDF file. PDF to JPEG conversion tools added so that 3rd party software is not required for this simple task
